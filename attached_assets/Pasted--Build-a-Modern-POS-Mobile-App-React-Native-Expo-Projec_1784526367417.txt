# Build a Modern POS Mobile App (React Native + Expo)

## Project Overview

Build a **production-quality Point of Sale (POS) mobile application** for Android and iOS using **React Native (Expo)** and **TypeScript**.

The application is designed for **small businesses** such as:

* Grocery Stores
* Mini Markets
* Clothing Stores
* Electronics Shops
* Pharmacies
* Bakeries
* Coffee Shops
* Hardware Stores

The app should be modern, extremely fast, scalable, and feel similar in quality to Shopify POS, Square POS, and Stripe Dashboard.

# Push Notifications (No SMS)

Do NOT implement SMS notifications.

Implement a complete in-app push notification system.

Requirements:

* Push notifications sent directly to the mobile app
* Real-time notifications
* Notification center inside the app
* Unread notification badge
* Mark as read
* Delete notification
* Notification history
* Different notification types

Examples:

New Order

Low Stock

Subscription Expiring

Employee Activity

New Customer

Inventory Transfer

Daily Sales Summary

System Announcement

Notification should open the related screen when tapped.

Use Firebase Cloud Messaging (FCM) or Expo Push Notifications.

# Final Goal

Build a premium, scalable POS application that feels like a commercial product rather than a demo. Every screen should be polished, responsive, and production-ready, with clean architecture and reusable components. The app should be easy to extend with backend APIs in the future.

This prompt gives Replit enough detail to generate a high-quality foundation while keeping the notification system entirely based on **mobile push notifications and an in-app notification center**, with no SMS functionality.
